***
WCP
***

.. module:: myokit.formats.wcp

Support is provided for reading data from files recorded using WinWCP_.

The :class:`WcpFile` class implements Myokit's shared
:class:`myokit.formats.SweepSource` interface.

.. _WinWCP: http://spider.science.strath.ac.uk/sipbs/showPage.php?page=software_ses
