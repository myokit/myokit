**********
Myokit IDE
**********

The Myokit IDE is an easy interface to Myokit. Models can be created or
imported from CellML and then modified, exported or used in simulations.

Implementation
==============
The gui is built using Qt, implemented in Python via either PyQt4 or PySide.
