.. _cmd/opencl:

**********
``opencl``
**********

Displays information about the current OpenCL configuration.

See :ref:`opencl-select <cmd/openclselect>` for information about setting a
preferred device.

Example::

    $ myokit opencl

    $ myokit opencl --help
